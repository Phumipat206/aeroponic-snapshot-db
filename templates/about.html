Aeroponic Snapshot Database

A web-based platform for managing plant growth snapshots from aeroponic systems. Capture, organize, search, and create time-lapse videos from your plant images.

Project Overview

This application was built for academic research in aeroponic plant cultivation. It provides a centralized system for collecting, storing, and analyzing plant growth images captured by cameras placed in aeroponic growing environments.

The platform supports both manual uploads via a web interface and automated capture using Raspberry Pi devices with scheduled crontab scripts. Images can be organized by project, camera, category, and time, making it easy to compare growth patterns and generate time-lapse videos.

Key Features

Upload & Import

Upload single images or bulk-import entire folders with automatic metadata detection.

Advanced Search

Filter by project, camera, category, tags, and date range with pagination.

Time-lapse Videos

Generate growth videos with customizable FPS and optional timestamp overlay.

REST API

Fully programmable API for automated uploads from IoT devices and scripts.

Smart Organization

Hierarchical categories, project/camera tagging, and auto-parsed folder structures.

Auto-Sync

Watch folders for new images and import them automatically in real-time.

IoT Integration

The system is designed for seamless integration with Raspberry Pi devices placed inside aeroponic growing chambers. Each Pi can be configured to automatically capture images at scheduled intervals and upload them via the REST API.
