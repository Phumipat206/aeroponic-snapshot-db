Auto-Sync (Folder Watcher)

Automatic folder monitoring system - Import images without pressing buttons

How It Works

The system monitors your designated folder. When new images appear, they are automatically imported.

Ideal For

Cameras that save images directly to folders, or systems requiring automation.

How to Use Folder Watcher

Command Line

python folder_watcher.py --watch "D:\CameraOutput" --category "Aeroponic System 1"

Options:

--watch, -w Folder to monitor
--category, -c Category name for imported images
--scan, -s Scan existing files before monitoring
--quiet, -q Quiet mode (show errors only)

Examples

# Monitor camera folder with existing file scan
python folder_watcher.py -w "E:\Photos" -c "Tray A" -s
# Monitor local folder
python folder_watcher.py -w "./input" -c "Test Category"

Supported File Formats

.jpg .jpeg .png .gif .bmp

Camera Setup

  1. Configure your camera to save images to a designated folder (e.g., D:\CameraOutput)
  2. Set up automatic capture intervals (e.g., every hour or daily at 9:00 AM)
  3. Run Folder Watcher to monitor that folder
  4. Images will be automatically imported whenever new ones appear

Tips

  • You can run multiple Folder Watchers simultaneously to monitor different folders
  • Use --scan to import existing files first
  • Categories are created automatically if they don't exist
  • Subfolder monitoring is supported (Recursive)
